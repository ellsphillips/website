# Demo content

This is a demo content file. It is used to show how to create a new content file in the `content` directory.

```jsx
import React from "react";

const Demo = () => {
  return (
    <div>
      <h1>Demo content</h1>
      <p>
        This is a demo content file. It is used to show how to create a new
        content file in the `content` directory.
      </p>
    </div>
  );
};

export default Demo;
```

## Usage

To create a new content file, you need to create a new file in the `content` directory. The file should have the `.mdx` extension.

```bash
content/
  stuff.mdx
```

The content file should have the following frontmatter:

```yaml
---
title: "Demo content"
layout: "project"
topic: "Personal"
description: ""
date: "2024-10-01"
---
```
